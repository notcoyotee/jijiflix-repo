Download Repository Addon

Installation Instructions:

  1. Download the repository addon using the button above
  2. In Kodi, go to: Add-ons → Install from zip file
  3. Select the downloaded ZIP file
  4. After installation, go to: Add-ons → Install from repository → JijiFlix Repository
  5. Select Video add-ons → JijiFlix and install
